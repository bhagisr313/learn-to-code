# 🚀 100 Programming Problems Roadmap (Loops & Conditions)

This repository contains a structured set of **100 programming problems** designed to strengthen fundamentals using:

- ✅ If Statements
- ✅ For Loops
- ✅ While Loops

Problems are grouped into **20 themes**, each with increasing difficulty to help build strong logic step-by-step.

---

## 🔹 THEME 1: Basic Printing Patterns

- Print numbers from 1 to N
- Print even numbers up to N
- Print numbers in reverse
- Print sum of first N numbers
- Print factorial of N

---

## 🔹 THEME 2: If Conditions Basics

- Check if number is positive/negative/zero
- Check even or odd
- Find greatest of two numbers
- Find greatest of three numbers
- Check leap year

---

## 🔹 THEME 3: Loop Control

- Print multiples of 3 up to N
- Skip multiples of 5
- Break when number divisible by 7

- Print first N numbers divisible by 4

---

## 🔹 THEME 4: Sum & Count Logic

- Sum of even numbers till N
- Count odd numbers till N
- Sum of digits of a number
- Count digits in number
- Product of digits

---

## 🔹 THEME 5: Number Reverse & Palindrome

- Reverse a number
- Check palindrome number

- Reverse only even digits
- Count palindromes in range

---

## 🔹 THEME 6: Factor & Divisibility

- Check prime number

- Sum of prime numbers till N

- Print multiplication table

## 🔹 THEME 8: Pattern Logic Advanced

## 🔹 THEME 10: While Loop Mastery

## 📌 How to Use This

- Start from Theme 1 and progress sequentially
- Solve at least **3–5 problems daily**
- Focus on **logic building**, not just syntax
- Revisit difficult problems after a few days

## 💡 Goal

By completing this roadmap, you will:

- Build strong problem-solving skills
- Master loops and conditions
- Prepare for coding interviews & real-world logic
